# GS01_MLAM
O presente projeto desenvolveu uma solução baseada em análise estatística de dados de missões espaciais com o objetivo de transformar dados operacionais em informações úteis para apoio à tomada de decisão.
<br><br>
Utilizando a linguagem Python e bibliotecas especializadas para análise de dados, foram construídas tabelas de distribuição de frequência, gráficos estatísticos e análises descritivas capazes de identificar padrões relacionados à carga útil transportada, custos das missões e composição das tripulações.
<br><br>
A solução permite que organizações do setor espacial obtenham uma visão consolidada de seus dados operacionais, facilitando a identificação de tendências, avaliação de desempenho e planejamento de futuras missões. A utilização de métricas estatísticas auxilia na redução de incertezas durante o processo de tomada de decisão e contribui para uma gestão mais eficiente dos recursos disponíveis.
<br><br>
Além disso, a metodologia aplicada pode ser expandida para bases de dados maiores e integradas a sistemas de monitoramento em tempo real, permitindo análises preditivas e geração de indicadores estratégicos para empresas e agências espaciais.
